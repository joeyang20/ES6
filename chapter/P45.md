# P45 45 ES6： 遍歷器Iterator

## 遍历器或者可遍历对象

```js
const colors = ['red','blue','green'];
//可以通过for of循环对可遍历对象进行循环
for(const color of colors){
    console.log(color);
}
//遍历器就是一个对象，这个对象有一个next方法
//可遍历对象就是具有Symbol.iterator属性的对象
const iterator = color[Symbol.iterator];
//每次调用next方法就可以返回下一个属性
iterator.next();
```

- 1.编写自己的遍历器

```js
const colors = ['red','blue','green'];
//返回数组的属性名和属性值对象
const iterator = color.entries();
iterator.next();
//keys方法也可以实现相同效果
const iterator = color.keys();
iterator.next();
//还有values方法,通过自己的方法来实现
Array.prototype.values = function(){
    let i = 0;
    let items = this;
    return {
        next(){
            const done = i>=items.length;
            const value = done ? undefined:items[i++];
            return {
                value,
                done
            }
        }
    }
}
const iterator = color.values();
iterator.next();
```
