# P54 54 ES6：WeakMap

## map和weakmap的不同

- 1.没有size属性
- 2.没有clear方法
- 3.不能被循环
- 4.key必须是对象
- 5.对象的引用被删除后，会通过垃圾回收机制清理掉对象的元素

```js
let jelly = {name:'jelly'};
let miffy = {name:'miffy'};

const strong = new Map();
const weak = new WeakMap();

strong.set(jelly,'jelly is the best!');
weak.set('miffy','miffy is the 2nd best!');
```

## 使用weakmap的场景

- 1.属性一定只能是对象的时候
- 2.当数据不可用，想要被回收内存的时候
